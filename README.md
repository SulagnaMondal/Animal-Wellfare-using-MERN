# Animal-Wellfare-using-MERN
A full stack application that allows users to browse pets for adoption, and upload their own pets.

MERN Stack (MongoDB, Express, React, Node.js) <br>
JWT Authentication & Bcrypt for secure user authentication <br>
MongoDB for user data storage with Mongoose ORM <br>
Role-based authentication (Admin/User) <br>
Node.js REST API Integration <br>
SCSS for styling <br>
Enhanced UI/UX with React-Modal, React-Router, and React-icons
